Variáveis e tipos básicos

Python não exige declarar tipos explícitos, e isso torna o código mais direto — mas também exige atenção ao manipular valores. Aqui estão alguns exemplos que costumo usar para lembrar como o Python infere tipos automaticamente.

nome = "Ana"
    idade = 23
    altura = 1.64
    estudando = True
    
    print(type(nome))
    print(type(idade))

Manipulação de Strings

Strings são muito usadas em Python, principalmente quando manipulamos dados, criamos mensagens, processamos arquivos ou tratamos entradas do usuário. Aqui estão as operações que mais aparecem no dia a dia.

Removendo espaços desnecessários

strip() remove espaços no começo e no fim da string. É útil para limpar dados de arquivos, logs ou textos gerados automaticamente.

texto = "  Olá, mundo!  "
    print(texto.strip())
    
    >> "Olá, mundo!"

Separando itens com split()

Quando uma string contém valores separados por vírgula, espaço ou outro delimitador, split() transforma o texto em uma lista.

frutas = "maçã,banana,uva"
    lista = frutas.split(",")
    
    >> ['maçã', 'banana', 'uva']

Concatenando textos

O jeito mais moderno e limpo de juntar textos é usando f-strings. É o formato mais legível e o que uso sempre.

saudacao = "Olá"
    destino = "mundo"
    
    mensagem = f"{saudacao}, {destino}!"
    print(mensagem)

Formatando mensagens com variáveis

Combinar textos com valores numéricos é muito comum. F-strings deixam isso claro e seguro.

nome = "Ana"
    idade = 23
    
    print(f"Meu nome é {nome} e eu tenho {idade} anos.")

Funções

Funções organizam o código, evitam repetição e tornam trechos reutilizáveis. Mesmo nos scripts mais simples, é sempre bom quebrar o código em funções. Aqui estão alguns exemplos básicos.

def saudacao(nome):
        return f"Olá, {nome}!"
    
    print(saudacao("Ana"))

Funções com argumentos padrão

Úteis quando queremos permitir que a função funcione mesmo sem receber todos os argumentos.

def somar(a, b=0):
        return a + b
    
    somar(5)    # 5
    somar(5, 7) # 12

List Comprehensions

Uma das construções mais úteis de Python. Permite criar listas novas a partir de outras listas, aplicando transformações ou filtragens, tudo em uma linha clara.

quadrados = [x**2 for x in range(5)]
    
    >> [0, 1, 4, 9, 16]

Com filtro:

pares = [n for n in range(10) if n % 2 == 0]
    >> [0, 2, 4, 6, 8]

Estruturas de Dados

Python oferece várias estruturas para armazenar coleções. Aqui estão exemplos que uso com frequência.

Listas

frutas = ["maçã", "banana", "uva"]
    frutas.append("mamão")

Dicionários

pessoa = {"nome": "Ana", "idade": 23}
    print(pessoa["nome"])

Tuplas (imutáveis)

coordenadas = (10, 20)
    coordenadas[0]

Sets (sem duplicatas)

numeros = {1, 2, 2, 3}
    >> {1, 2, 3}

Tratamento de Erros

Quando algo pode dar errado — conversão, leitura de arquivo, entrada do usuário — usamos try/except para evitar que o programa quebre.

try:
        numero = int("abc")
    except ValueError:
        print("Erro: valor inválido")

Argumentos de Linha de Comando

Python permite receber argumentos diretamente do terminal, o que é útil para scripts.

sys.argv

import sys
    print(sys.argv)

argparse

Quando queremos algo mais organizado, usamos argparse:

import argparse
    
    parser = argparse.ArgumentParser()
    parser.add_argument("nome")
    parser.add_argument("--idade", default=25)
    args = parser.parse_args()
    
    print(args.nome, args.idade)

Datas e Horários

Trabalhar com datas é muito comum quando lidamos com relatórios, logs ou organização temporal.

from datetime import datetime
    
    agora = datetime.now()
    print(agora.strftime("%d/%m/%Y %H:%M"))

JSON

JSON é um formato muito usado para armazenar e trocar dados. Python oferece suporte nativo.

Objeto → JSON

import json
    
    dados = {"nome": "Ana", "idade": 23}
    json_str = json.dumps(dados)

JSON → Objeto

obj = json.loads(json_str)
    print(obj["nome"])

Requests

Requests é uma das bibliotecas mais populares para fazer requisições HTTP.

import requests
    
    r = requests.get("https://api.github.com")
    print(r.status_code)

Manipulação de Arquivos

A leitura e escrita de arquivos é uma das tarefas mais frequentes em scripts.

Lendo arquivos

with open("dados.txt") as f:
        print(f.read())

Escrevendo arquivos

with open("saida.txt", "w") as f:
        f.write("Python é incrível!")

Caminhos e arquivos (pathlib)

pathlib torna muito mais fácil manipular caminhos e arquivos, especialmente no Linux.

from pathlib import Path
    
    p = Path("/home/ana")
    print(p.exists())

Rodar comandos do sistema

Útil para chamar comandos externos dentro de scripts Python.

import subprocess
    subprocess.run(["ls", "-l"])
