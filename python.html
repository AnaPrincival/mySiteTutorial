Manipulação de Strings

Strings são sequências de caracteres. Python fornece diversos métodos úteis para manipulá-las.

Removendo espaços e separando palavras

texto = "  Olá, mundo!  "
print(texto.strip())  # Remove espaços extras

>> "Olá, mundo!"
frase = "maçã,banana,uva"
print(frase.split(","))  # Divide a string por vírgula
 >> ['maçã', 'banana', 'uva']

Concatenando Strings

parte1 = "Olá"
parte2 = "Mundo"
mensagem = parte1 + ", " + parte2 + "!"
print(mensagem)
 >> Olá, Mundo!

Formatando Strings

nome = "Ana"
idade = 23
print(f"Meu nome é {nome} e eu tenho {idade} anos.")
 >> Meu nome é Ana e eu tenho 23 anos.

Argumentos de Linha de Comando

Podemos capturar argumentos passados pelo terminal usando sys.argv.

import sys
print("Argumentos recebidos:", sys.argv)
$ >> python script.py arg1 arg2
Argumentos recebidos: ['script.py', 'arg1', 'arg2']

Usando argparse para gerenciar argumentos

O módulo argparse é mais poderoso e permite definir argumentos opcionais e obrigatórios de forma clara.

import argparse

parser = argparse.ArgumentParser(description="Exemplo de argparse")
parser.add_argument("nome", type=str, help="Nome do usuário")
parser.add_argument("--idade", type=int, help="Idade do usuário", default=25)
args = parser.parse_args()

print(f"Nome: {args.nome}")
print(f"Idade: {args.idade}")
$ >> python script.py Ana --idade 23
Nome: Ana
Idade: 23

Loops

Repetição de blocos de código em Python.

Loop For

for i in range(3):
    print(f"Iteração {i}")
 >> Iteração 0
Iteração 1
Iteração 2

Loop While

contador = 0
while contador < 3:
    print(f"Contagem: {contador}")
    contador += 1
 >> Contagem: 0
Contagem: 1
Contagem: 2

Manipulação de Arquivos

Abrindo, lendo e escrevendo arquivos em Python.

Lendo Arquivos

with open("dados.txt", "r") as arquivo:
    print(arquivo.read())
 >> Olá, Mundo!

Escrevendo em Arquivos

with open("saida.txt", "w") as arquivo:
    arquivo.write("Python é incrível!\n")

O arquivo saida.txt conterá:

Python é incrível!
