Variáveis e tipos básicos

Python permite criar variáveis sem declarar tipo explicitamente.

nome = "Ana"
    idade = 23
    altura = 1.64
    estudando = True
    
    print(type(nome))   # 
    print(type(idade))  #

Manipulação de Strings

Strings são sequências de caracteres usadas para representar textos.

Strip e Split

texto = "  Olá, mundo!  "
    print(texto.strip())
    
    >> "Olá, mundo!"
fruta_str = "maçã,banana,uva"
    lista = fruta_str.split(",")
    
    >> ['maçã', 'banana', 'uva']

Concatenação e f-strings

saudacao = "Olá"
    destino = "mundo"
    
    mensagem = f"{saudacao}, {destino}!"
    print(mensagem)
    
    >> Olá, mundo!

Funções

def saudacao(nome):
        return f"Olá, {nome}!"
    
    print(saudacao("Ana"))

Argumentos padrão

def somar(a, b=0):
        return a + b
    
    print(somar(5))   # 5
    print(somar(5, 7))  # 12

List Comprehensions

Uma forma elegante e rápida de criar listas.

quadrados = [x**2 for x in range(5)]
    >> [0, 1, 4, 9, 16]
pares = [n for n in range(10) if n % 2 == 0]
    >> [0, 2, 4, 6, 8]

Estruturas de Dados

Listas

frutas = ["maçã", "banana", "uva"]
    frutas.append("mamão")
    
    >> ['maçã', 'banana', 'uva', 'mamão']

Dicionários

pessoa = {"nome": "Ana", "idade": 23}
    print(pessoa["nome"])
    
    >> Ana

Tuplas

coordenadas = (10, 20)
    coordenadas[0]
    >> 10

Sets

numeros = {1, 2, 2, 3}
    >> {1, 2, 3}
