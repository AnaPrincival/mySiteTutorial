Introdução ao Jinja

Jinja é um mecanismo de template poderoso para Python, permitindo a geração dinâmica de HTML.

Uso de Variáveis

Podemos exibir valores dinâmicos diretamente no template:

{{ nome }}

Saída (se nome="Ana"):

Ana

Estruturas de Controle

Usando IF/ELSE

{% if idade >= 18 %}
    Você é maior de idade.
{% else %}
    Você é menor de idade.
{% endif %}

Saída (se idade=20):

Você é maior de idade.

Usando FOR

{% for fruta in frutas %}
    {{ fruta }}
{% endfor %}

Saída (se frutas=['maçã', 'banana', 'uva']):

maçã
banana
uva

Filtros no Jinja

Jinja possui filtros para modificar variáveis diretamente no template.

{{ nome | upper }}

Saída (se nome="ana"):

ANA

Blocos e Herança

Jinja permite herança de templates, reutilizando estruturas comuns.

{% extends "base.html" %}

{% block conteudo %}
    
Bem-vindo!

{% endblock %}
