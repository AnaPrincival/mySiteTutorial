Manipulação de Strings

Strings são sequências de caracteres. Python oferece métodos simples e poderosos para analisar, limpar e transformar textos.

Removendo espaços e separando palavras

texto = "  Olá, mundo!  "
    print(texto.strip())   # Remove espaços antes e depois
    
    >> "Olá, mundo!"
fruta_str = "maçã,banana,uva"
    lista = fruta_str.split(",")   # Separa pela vírgula
    print(lista)
    
    >> ['maçã', 'banana', 'uva']

Concatenando strings

saudacao = "Olá"
    destino = "mundo"
    
    mensagem = f"{saudacao}, {destino}!"
    print(mensagem)
    
    >> Olá, mundo!

Formatando strings com f-strings

nome = "Ana"
    idade = 23
    
    print(f"Meu nome é {nome} e eu tenho {idade} anos.")
    
    >> Meu nome é Ana e eu tenho 23 anos.

Argumentos de Linha de Comando

Usamos módulos como sys e argparse para receber valores passados pelo terminal.

Capturando argumentos com sys.argv

import sys
    
    print("Argumentos recebidos:", sys.argv)
    
    $ python script.py arg1 arg2
    >> Argumentos recebidos: ['script.py', 'arg1', 'arg2']

Gerenciando argumentos com argparse

argparse permite criar argumentos obrigatórios e opcionais de forma organizada.

import argparse
    
    parser = argparse.ArgumentParser(description="Exemplo de argparse")
    parser.add_argument("nome", type=str, help="Nome da pessoa")
    parser.add_argument("--idade", type=int, default=25, help="Idade opcional")
    args = parser.parse_args()
    
    print(f"Nome: {args.nome}")
    print(f"Idade: {args.idade}")
    
    $ python script.py Ana --idade 23
    >> Nome: Ana
    >> Idade: 23

Estruturas de Dados

Python oferece estruturas versáteis para armazenar coleções de valores.

Listas

frutas = ["maçã", "banana", "uva"]
    frutas.append("mamão")
    print(frutas)
    
    >> ['maçã', 'banana', 'uva', 'mamão']

Dicionários

pessoa = {"nome": "Ana", "idade": 23}
    print(pessoa["nome"])
    
    >> Ana

Tuplas (imutáveis)

coordenadas = (10, 20)
    print(coordenadas[0])
    
    >> 10

Sets (não ordenados e sem duplicatas)

numeros = {1, 2, 2, 3}
    print(numeros)
    
    >> {1, 2, 3}

Loops

Loops permitem repetir ações enquanto uma condição for verdadeira.

Loop for

for i in range(3):
        print(f"Iteração {i}")
    
    >> Iteração 0
    >> Iteração 1
    >> Iteração 2

Loop while

contador = 0
    
    while contador < 3:
        print(contador)
        contador += 1
    
    >> 0
    >> 1
    >> 2
