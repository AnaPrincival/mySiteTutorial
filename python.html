Manipulação de Strings

Strings são sequências de caracteres. Python fornece métodos úteis para manipulá-las.

strip e split

texto = "  Olá, mundo!  "
print(texto.strip())  # "Olá, mundo!"

texto = "  Python   "
print(texto.strip(" "))  # "Python"

frase = "maçã,banana,uva"
print(frase.split(","))  # ["maçã", "banana", "uva"]

frase = "um dois três quatro"
print(frase.split())  # ["um", "dois", "três", "quatro"]

linha = "Nome: Ana; Idade: 23; Cidade: Caen"
print(linha.split("; "))  # ['Nome: Ana', 'Idade: 23', 'Cidade: Caen']

Loops

Loops são usados para repetir um bloco de código várias vezes.

Loop for

for i in range(5):
    print(f"Iteração {i}")

lista = ["maçã", "banana", "uva"]
for fruta in lista:
    print(fruta)

for letra in "Python":
    print(letra)

for i in range(2, 10, 2):
    print(i)  # Números pares de 2 a 8

usuarios = {"Ana": 23, "Carlos": 30}
for nome, idade in usuarios.items():
    print(f"{nome} tem {idade} anos")

Loop while

contador = 0
while contador < 3:
    print(f"Contagem: {contador}")
    contador += 1

numero = 10
while numero > 0:
    print(numero)
    numero -= 2

resposta = ""
while resposta.lower() != "sim":
    resposta = input("Deseja continuar? (sim/não): ")

from random import randint
sorteado = randint(1, 10)
chute = 0
while chute != sorteado:
    chute = int(input("Adivinhe o número: "))
    print("Errou, tente novamente!")

Manipulação de Arquivos

O Python permite abrir, ler e escrever arquivos de forma simples.

Abrindo e lendo arquivos

with open("exemplo.txt", "r") as arquivo:
    print(arquivo.read())  # Lê todo o conteúdo do arquivo

with open("exemplo.txt", "r") as arquivo:
    linhas = arquivo.readlines()  # Retorna uma lista de linhas
    print(linhas)

with open("exemplo.txt", "r") as arquivo:
    for linha in arquivo:
        print(linha.strip())  # Remove espaços e quebras de linha

Escrevendo em arquivos

with open("saida.txt", "w") as arquivo:
    arquivo.write("Primeira linha\n")
    arquivo.write("Segunda linha\n")

with open("saida.txt", "a") as arquivo:
    arquivo.write("Linha adicionada no final\n")
