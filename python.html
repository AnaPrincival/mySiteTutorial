Manipulação de Strings

strip e split

O método strip() remove espaços em branco do início e fim de uma string.

texto = "  Olá, mundo!  "
print(texto.strip())  # "Olá, mundo!"

O método split() divide uma string em uma lista de substrings com base em um delimitador.

frase = "maçã,banana,uva"
print(frase.split(","))  # ["maçã", "banana", "uva"]

Argumentos de Linha de Comando

argparse

O módulo argparse é usado para processar argumentos passados na linha de comando.

import argparse

parser = argparse.ArgumentParser(description="Exemplo de argparse")
parser.add_argument("nome", type=str, help="Nome do usuário")
args = parser.parse_args()
print(f"Olá, {args.nome}!")

Estruturas de Dados

Tuplas

Tuplas são imutáveis e são definidas com parênteses.

cores = ("vermelho", "azul", "verde")
print(cores[0])  # vermelho

Listas

Listas são mutáveis e permitem adição e remoção de elementos.

numeros = [1, 2, 3]
numeros.append(4)
print(numeros)  # [1, 2, 3, 4]

Dicionários

Dicionários armazenam pares chave-valor.

dados = {"nome": "Ana", "idade": 23}
print(dados["nome"])  # Ana
