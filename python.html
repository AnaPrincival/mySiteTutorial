Manipulação de Strings

Strings são sequências de caracteres. Python fornece diversos métodos úteis para manipulá-las.

Removendo espaços e separando palavras

texto = "  Olá, mundo!  "
print(texto.strip())  # Remove espaços extras

frase = "maçã,banana,uva"
print(frase.split(","))  # Divide a string por vírgula

Concatenando Strings

parte1 = "Olá"
parte2 = "Mundo"
mensagem = parte1 + ", " + parte2 + "!"
print(mensagem)

Formatando Strings

nome = "Ana"
idade = 23
print(f"Meu nome é {nome} e eu tenho {idade} anos.")

Argumentos de Linha de Comando

Podemos capturar argumentos passados pelo terminal usando sys.argv.

import sys
print("Argumentos recebidos:", sys.argv)

Estruturas de Dados

Python possui diversas estruturas para armazenar e manipular coleções de dados.

Listas

lista = [1, 2, 3, 4]
lista.append(5)  # Adiciona um item
print(lista)

# Percorrendo uma lista
for item in lista:
    print(item)

Tuplas

tupla = (10, 20, 30)
print(tupla[1])  # Acessando um elemento

Dicionários

usuario = {"nome": "Ana", "idade": 23}
print(usuario["nome"])
usuario["cidade"] = "Caen"
print(usuario)

Conjuntos

conjunto = {1, 2, 3, 4}
conjunto.add(5)
print(conjunto)

Loops

Repetição de blocos de código em Python.

Loop For

for i in range(5):
    print(f"Iteração {i}")

Loop While

contador = 0
while contador < 5:
    print(f"Contagem: {contador}")
    contador += 1

Manipulação de Arquivos

Abrindo, lendo e escrevendo arquivos em Python.

Lendo Arquivos

with open("dados.txt", "r") as arquivo:
    print(arquivo.read())

Escrevendo em Arquivos

with open("saida.txt", "w") as arquivo:
    arquivo.write("Olá, mundo!\n")
