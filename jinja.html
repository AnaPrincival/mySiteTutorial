Introdução ao Jinja

Jinja é uma linguagem de template moderna para Python. Ele permite criar HTML dinâmico, facilitando a separação entre lógica de programação e apresentação visual.

Uso de Variáveis

No Jinja, podemos exibir variáveis dinamicamente dentro dos templates:

{{ usuario }}

Saída (se usuario = "Ana"):

Ana

Estruturas de Controle

IF/ELSE

{% if idade >= 18 %}
    Você pode acessar o site.
{% else %}
    Acesso negado.
{% endif %}

Saída (se idade = 20):

Você pode acessar o site.

FOR Loop

{% for item in lista %}
    {{ item }}
{% endfor %}

Saída (se lista = ["maçã", "banana", "uva"]):

maçã
banana
uva

Filtros

Filtros são usados para modificar valores no Jinja.

Converter para maiúsculas

{{ nome | upper }}

Saída (se nome = "ana"):

ANA

Formatar número

{{ preco | round(2) }}

Saída (se preco = 5.6789):

5.68

Macros

Macros são como funções dentro dos templates, permitindo reutilização de código.

{% macro saudacao(nome) %}
    Olá, {{ nome }}!
{% endmacro %}

{{ saudacao("Ana") }}

Saída:

Olá, Ana!

Blocos e Herança

O Jinja suporta herança de templates, permitindo reutilização de layouts.

{% extends "base.html" %}

{% block conteudo %}
    
Bem-vindo ao site!

{% endblock %}
