About Python

Python is a high-level programming language created in the early 1990s by Guido van Rossum, who designed it with the goal of making code both readable and pleasant to write. Over the years, it has grown into one of the most widely used languages in research, industry, and education, thanks to its clean syntax and extensive ecosystem of libraries. Python is not a compiled language in the traditional sense; instead, it is interpreted by the CPython runtime, which translates the code at execution time. This gives developers a fast development cycle, even though it may not reach the raw performance of low-level compiled languages like C or C++. The language manages memory automatically through a built-in garbage collector that tracks and frees unused objects, allowing programmers to focus more on logic than on manual memory control. Its versatility—ranging from scripting and automation to data science, machine learning, and backend services—makes Python a flexible tool for beginners and advanced users alike.

Variables and basic types

Python does not require explicit type declarations. This makes the code more direct, but also means we need to pay attention to the values being handled. Here are some examples showing how Python infers types automatically.


    name = "Ana"
    age = 23
    height = 1.64
    studying = True
    
    print(type(name))   # <class 'str'>
    print(type(age))    # <class 'int'>

String Manipulation

Strings are extremely common in Python, especially when working with text, logs, user input, or data cleaning. Here are the operations I use the most.

Removing extra spaces

strip() removes whitespace at the beginning and end of the string. Very useful when processing data from files or user input.


    text = "  Hello, world!  "
    print(text.strip())
    
    # Output:
    # "Hello, world!"

Splitting strings

split() transforms a string into a list, based on a separator.


    fruits = "apple,banana,grape"
    lst = fruits.split(",")
    
    print(lst)
    # Output:
    # ['apple', 'banana', 'grape']

Concatenating strings

The cleanest and most modern way to combine text in Python is using f-strings.


    greeting = "Hello"
    target = "world"
    
    message = f"{greeting}, {target}!"
    print(message)
    
    # Output:
    # Hello, world!

Formatting messages with variables

Very useful when mixing text and numbers.


    name = "Ana"
    age = 23
    
    print(f"My name is {name} and I am {age} years old.")
    
    # Output:
    # My name is Ana and I am 23 years old.

Functions

Functions help organize code, avoid repetition, and make logic reusable. Even in simple scripts, it's worth breaking tasks into functions. Here are some basic examples.


    def greet(name):
    return f"Hello, {name}!"
    
    print(greet("Ana"))
    
    # Output:
    # Hello, Ana!

Default arguments

Useful when we want a function to work even without all arguments provided.


    def add(a, b=0):
    return a + b
    
    print(add(5))      # 5
    print(add(5, 7))   # 12

List Comprehensions

One of Python’s most powerful constructs. It lets us create lists in a clean and expressive way.


    squares = [x**2 for x in range(5)]
    print(squares)
    
    # Output:
    # [0, 1, 4, 9, 16]

With a filter:


    evens = [n for n in range(10) if n % 2 == 0]
    print(evens)
    
    # Output:
    # [0, 2, 4, 6, 8]

Data Structures

Python offers several ways to store collections of values.

Lists

Lists are mutable, ordered collections. You can add, remove, or modify elements.


    fruits = ["apple", "banana", "grape"]
    fruits.append("papaya")
    print(fruits)
    
    # Output:
    # ['apple', 'banana', 'grape', 'papaya']

Accessing and modifying elements:


    fruits[1] = "kiwi"
    print(fruits)
    
    # Output:
    # ['apple', 'kiwi', 'grape', 'papaya']

Useful operations:


    print(len(fruits))      # 4
    print(fruits[:2])       # slicing → ['apple', 'kiwi']
    print("grape" in fruits)  # True

Dictionaries

Dictionaries map keys to values. They are mutable and extremely useful.


    person = {"name": "Ana", "age": 23}
    print(person["name"])
    
    # Output:
    # Ana

Adding or modifying keys:


    person["city"] = "Grenoble"
    person["age"] = 24
    print(person)
    
    # Output:
    # {'name': 'Ana', 'age': 24, 'city': 'Grenoble'}

Iterating over keys and values:


    for key, value in person.items():
        print(key, value)
    
    # Output:
    # name Ana
    # age 24
    # city Grenoble

Tuples (immutable)

Tuples are ordered and immutable. Great for values that shouldn't change.


    coords = (10, 20)
    print(coords[0])
    
    # Output:
    # 10

Tuple unpacking:


    x, y = coords
    print(x, y)
    
    # Output:
    # 10 20

Sets (no duplicates)

Sets remove duplicates automatically and allow fast membership checks.


    numbers = {1, 2, 2, 3}
    print(numbers)
    
    # Output:
    # {1, 2, 3}

Common operations:


    a = {1, 2, 3}
    b = {3, 4, 5}
    
    print(a.union(b))        # {1, 2, 3, 4, 5}
    print(a.intersection(b)) # {3}
    print(a - b)             # {1, 2}

Fast membership check:


    print(3 in a)   # True
    print(10 in a)  # False

Error Handling

When something can fail — reading a file, converting a value, user input — we use try/except to prevent the script from crashing.


        try:
        number = int("abc")
    except ValueError:
        print("Error: invalid value")
    
    # Output:
    # Error: invalid value

Command-Line Arguments

Python can read arguments directly from the terminal, which is useful for scripts that need input values.

sys.argv


    import sys
    print(sys.argv)
    
    # Output example:
    # ['script.py', 'arg1', 'arg2']

argparse

When we want organized and documented arguments, we use argparse:


    import argparse
    
    parser = argparse.ArgumentParser()
    parser.add_argument("name")
    parser.add_argument("--age", default=25)
    args = parser.parse_args()
    
    print(args.name, args.age)
    
    # Example output:
    # Ana 25

Dates and Time

Working with dates is common when generating reports, managing logs, or dealing with timestamps.


    from datetime import datetime
    
    now = datetime.now()
    print(now.strftime("%d/%m/%Y %H:%M"))
    
    # Example output:
    # 15/11/2025 18:42

JSON

JSON is widely used to store and exchange structured data. Python supports it natively.

Object → JSON


    import json
    
    data = {"name": "Ana", "age": 23}
    json_str = json.dumps(data)
    print(json_str)
    
    # Example output:
    # {"name": "Ana", "age": 23}

JSON → Object


    obj = json.loads(json_str)
    print(obj["name"])
    
    # Output:
    # Ana

Requests

Requests is one of the most popular libraries for making HTTP requests.


    import requests
    
    r = requests.get("https://api.github.com")
    print(r.status_code)
    
    # Example output:
    # 200

File Handling

Reading and writing files is one of the most common tasks in scripts.

Reading files


    with open("data.txt") as f:
    print(f.read())

    # Output example:
    # (contents of data.txt)

Writing files


    with open("output.txt", "w") as f:
    f.write("Python is amazing!")
    
    # After running, output.txt contains:
    # Python is amazing!

Paths and files (pathlib)

pathlib makes it easier to handle file paths, especially on Linux.


    from pathlib import Path
    
    p = Path("/home/ana")
    print(p.exists())
    
    # Output example:
    # True

Running System Commands

Useful when calling external programs within a Python script.


    import subprocess
    subprocess.run(["ls", "-l"])
    
    # Output example:
    # total 8
    # -rw-r--r-- 1 ana ana  120 Nov 15  file.txt
    # drwxr-xr-x 1 ana ana 4096 Nov 15  folder/
